# 项目基本依赖
- 开发工具: goland 2019.3候选版本 (https://download.jetbrains.8686c.com/go/goland-193.5233.69.exe)
- sdk版本 : 1.13.4
- 代理辅助: https://goproxy.cn

# 使用的具体技术 
- 配置文件: yaml
- 缓存技术: redis
- 存储技术: mysql
- 消息队列: rabbitmq
- MVC框架 : gin
- rpc框架 : grpc
- 注册发现: etcd
- 链路监控: jaeger
- 应用监控: prometheus
- 定时任务: cron+etcd
